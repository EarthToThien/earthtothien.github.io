Introduction

This is a detailed tutorial on how to make a PNG avatar in OBS. This tutorial will show how to link the avatar to the streamer's microphone, but other audio sources may be used.

Requirements

  • OBS Plugins
    • Move
  • Assets
    • Idle Image
    • Talking Image

Installation

Create a new scene in OBS for the PNG avatar. The scene for the PNG avatar will be added to whatever scene the avatar will appear using nested scenes. In the newly created avatar scene, create an image source the Idle Image and another for the Talking Image. Layer the talking image over the idle image (Figure 1). The idea is to have OBS swap the visibility of the two sources while the streamer is speaking.

In the Audio Mixer for OBS, open the Filters window for the microphone or some other dedicated audio source. Right click in the Audio Filters box, and if the Move plugin is installed correctly, a filter called Audio Move should be available (Figure 2).

FIGURE 2 HERE

Add an Audio Move filter and name it "Talking". Set Action for the filter to "Source Visibility." For Set the avatar scene to the avatar scene, and set the source to the Talking Image source. For Threshold Action, set to "Enable Over and Disable Under." As a reference, Figure 3 shows the settings for the other options. Easing and Threshold can be adjusted to preference. Repeat these steps for the Idle Image, but for Threshold Action, set to "Enable Under and Disable Over."
