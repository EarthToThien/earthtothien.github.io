Introduction

This is a detailed tutorial on how to make a PNG avatar using only OBS and the Move plugin by Exeldro. This tutorial will show how to setup the source assets for the avatar in OBS and link it to the streamer's microphone. Alternatively, other audio sources may be used, and the steps required are the same.

Requirements

  • OBS Plugins
    • Move
  • Assets
    • Idle Image
    • Talking Image

Installation

Create a new scene in OBS for the PNG avatar. This scene will be added to other scenes where the avatar will appear via nested scenes. In the newly created avatar scene, create an image source for the Idle Image and another for the Talking Image. Layer the talking image over the idle image (Figure 1). The idea here will be to have OBS swap the visibility of the two sources while the streamer is speaking. It is recommended that these images have subtle movements (ie mouth opening and closing) for the best effect, but for this tutorial, the images selected have large movements to demonstrate the swapping of source visibilities.

In the Audio Mixer for OBS, open the Filters window for the microphone source or alternate dedicated audio source. Right click in the Audio Filters box, and if the Move plugin is installed correctly, a filter called Audio Move should be available (Figure 2).

Add an Audio Move filter and name it "Talking". Set Action for the filter to "Source Visibility." Set the avatar scene as the designated Scene, and set the Talking Image source for the designated Source. For Threshold Action, set to "Enable Over and Disable Under." As a reference, Figure 3 shows the recommended settings for the additional options. Easing and Threshold can be adjusted to preference and microphone setup. Repeat these steps for the Idle Image source, but for Threshold Action, set to "Enable Under and Disable Over." As the streamer speaks, the Talking Image will be visible while the Idle Image will be hidden and vice versa when the streamer is silent. As previously stated, it is recommended that there are subtle movements between the two sources such as the mouth opening and closing to have the best effect with the PNG avatar.
