Thien's Current OBS Plugin List

Visual Effects

  • 3D Effect - Adds a 3D effect filter to OBS sources. Effect can be observed in my Starting and Ending scene.
  • Advanced Masks - Allows me to apply different mask filters to sources in OBS. Some example of how I use this plugin is to apply different shapes to my camera for effects such as the Anime Reaction channel point redeem.
  • Composite Blur - Adds different blur effects to OBS sources. Effect can be observed during coding streams when the desktop capture needs to be obscurred to hide private or sensitive information. Other examples of usage include the Clean Your Glasses channel point redeem where parts of the scene outside the glasses are blurred giving the effect of glasses being removed.
  • Freeze Filter - Freezes the frame of a source. The plugin is used in photo-like effects such as the Polaroid and Motivation channel point redeem.
  • Gradient Source - Adds a gradient color source.
  • Markdown Source - Alternate text source in OBS. The plugin allows the user to type in markdown which can be converted to html and displayed using a browser source. Custom CSS can be applied to the source. Example of how the plugin is used can be observed in the Tasklist widget.
  • Move - This is THE plugin to have installed. It is the most versatile tool in my collection, and I have it integrated in almost every aspect of my OBS setup. At face value, it allows me to create motion effects for my sources (ie move a source around the scene), but there are additional filters included in the plugin such as Move Value and Audio Move that add more functionality.
  • OBS Shaderfilter - This plugin is a collection of user-created filters. These effects include rounded corners for sources, rainbow color filter, and rotating source.
  • Retro Effects - Adds different retro effect filters such as Digital Glitch or Frame Skip to OBS sources. Effect can be observed in the Crash the Stream channel point redeem as the camera freezes and glitches before the crash animation.
  • Scale to Sound - This is a reactive audio plugin. Adds a filter that shrinks and grows OBS sources based on the applied playing audio.
  • Source Clone - OBS source plugin that allows me to create a duplicate of an existing source at equal or lower quality. I am able to manipulate this source and apply filters without affecting the original source.
  • Stroke Glow Shadow - This plugin adds effect filters to OBS for applying stroke, glow, and drop shadow to any source.
  • Waveform - OBS source plugin to generate customized audio visual waveforms that react to your audio sources.

Functional

  • Downstream Keyer - Creates global scenes. For example, I am able to add a scene that contains all of my Twitch alerts and set it as a layer over all of my other scenes in my OBS setup.
  • jrDockie - This plugin is an organizational tool that helps reduce clutter in my streaming setup, allowing me to create presets for my docks and their positions in the OBS window.
  • Quick Access Utility - This custom dock plugin adds functionality and improves workflow of my OBS setup. Along with the jrDockie plugin, I am able to create custom docks such as one that lists only my live production scenes. I have also created search dock that allows me to do a quick search for specific sources and filters. Now I don't need to scroll through my entire Scenes list and click on every source to look for a filter.
  • Source Copy - Import/Export OBS scenes including sources and filters.
  • StreamUP OBS Studio Plugin - Adds extra functionality to OBS. It checks if all plugins in my OBS installation are up to date and allows me to refresh all my browser sources in one action.
  • Transition Table - Organizational plugin allowing me to set specific transitions between my scenes.
